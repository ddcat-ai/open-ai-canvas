# CHANGELOG

## Unreleased

- 管理员可从上游渠道一键拉取模型目录；缺失模型自动导入为停用待定价，避免浏览器直连上游时的 CORS 问题。

- 系统后台支持热更新 Worker 与全局渠道并发，各系统渠道可跟随全局值或单独覆盖；渠道满载时请求等待空闲槽位，不再直接失败。

## v1.0.3

- 系统公告正文支持受控 HTML，用户可以看到蓝色链接并在新窗口打开，同时支持粗体、斜体、下划线、列表和换行等常见格式。
- 修复本地 Codex 会话新建后首次发送失败的问题。

## v1.0.2

- 增加前端与后端多架构 GHCR 镜像发布，并让服务器一键部署直接拉取指定版本镜像。
- 完善 Codex App 插件的仓库 marketplace 安装入口。
- 修复 xAI 视频生成请求未使用官方字段的问题。

## v1.0.1

- 自定义模型渠道改由登录态后端中转，减少第三方接口 CORS 限制。
- 增加 xAI 与 Sub2API 的 Grok 视频生成协议支持。
- 增加私有上游精确主机白名单，收紧自定义渠道访问边界。
- 修复 Agent 会话未展示后端具体失败原因的问题。
- 修复主页删除最近项目时未显示确认弹窗的问题。
- 修复 PostgreSQL 用户活跃统计冲突更新时的列歧义问题。

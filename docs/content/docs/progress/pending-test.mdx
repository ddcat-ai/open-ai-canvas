# 待测试

- 开源发布前命名与仓库边界清理：PostgreSQL 默认库名和用户、SQLite 文件名、会话 Cookie 与前端配置存储键统一使用 `open_ai_canvas`；两类专用视频协议改为 `NewAPI 渠道 1/2` 和 `newapi-channel-1/2`，前后端、错误文案与测试不再包含原渠道名，两种渠道均不预填 Base URL；根级忽略规则新增 `.local/` 与 Claude 本地配置，并允许新仓库纳入 `docs/` 源码。需验证全新 SQLite/PostgreSQL 部署、登录 Cookie、配置保存、三种 NewAPI 视频渠道创建与任务分发、渠道切换后 Base URL 为空，以及全新仓库 `git add .` 不包含数据库、部署日志和本机配置。

- 模型可选项清空与失效清理：图片、视频、文本和音频模型可选项显式清空后持久保存空数组，重新登录、刷新页面及系统渠道刷新时不再自动补回推荐项；后台停用或删除模型后，用户配置同步移除对应可选项并重新选择有效默认模型，同时清理历史本地配置中的空值、`undefined` 和 `null` 模型名。需验证单项删除、全部清空、刷新页面、退出重登、系统模型刷新、后台停用当前默认模型、同渠道仍有其他模型和渠道全部模型停用。

- 首页主操作按钮主题色：按钮背景、边框和文字使用明确的中性黑白配色，浅色主题保持深底浅字，深色主题切换为浅底深字；需验证首次加载、连续切换主题、禁用态及悬浮态的文字对比度。

- 画布拖拽、缩放与主题切换性能：节点拖拽开始时一次性构建对齐锚点，鼠标移动只记录最新坐标并在动画帧内计算，避免高刷新率鼠标重复执行全节点扫描；拖拽期间统一关闭节点 Comet 3D 表面。画布缩放和平移期间降低浮层定位通知频率，并暂时关闭节点阴影过渡、生成描边和连线流光，结束后恢复最终视口；自动性能模式在 80 个节点或 32 个媒体节点时提前启用。画布页面切换亮暗主题不再创建整页 View Transition 快照，改为单次同步更新；普通页面主题动画增加防重入和显式清理。需在 Windows Chrome/Edge 与 macOS Chrome 上验证鼠标拖拽、框选后多节点拖拽、背板及批次子节点联动、对齐辅助线、拖入背板、滚轮缩放、触控板平移/缩放、连续快速切换亮暗主题、减少动态效果，以及约 30/80/160 个节点时的 CPU、帧率、DOM 数量和交互结束后的阴影/连线恢复。

- 画布媒体替换与创作管理：图片、视频卡片悬浮时在底部约 10% 位置只显示单层胶囊“替换”按钮，单击媒体卡片打开提示词编辑；替换文件保留节点 ID、位置、尺寸、连线及管理元数据，沿用上传进度反馈和画布撤销历史，失败不覆盖原媒体；文件拖到同类型卡片或单选图片后粘贴剪切板图片可直接替换，否则继续创建新节点。已有图片原位重生成会保留生成前快照，视频连续生成会进入同一 A/B/C 版本族，可在版本对比中指定唯一主版本；节点支持位置/尺寸锁定。顶部支持搜索标题、提示词、模型和标签并定位节点，提供自动/画质优先/性能优先三档媒体性能模式；多选媒体可创建自动宫格引用组，折叠后沿用背板的聚合连线路由，并继续支持按连线整理。需验证图片/视频悬浮与触屏选中、单击打开提示词、替换按钮不触发节点拖动、文件选择取消、上传失败、拖放命中重叠节点、剪切板权限、撤销重做、生成失败/重试/多图批次、主版本切换、锁定后拖动/缩放/生成、搜索无结果与 Ctrl/Cmd+K、32 个媒体或 80 个总节点的自动性能阈值、三档模式持久化、引用组折叠/展开/移动/删除、导入导出和账号同步。

- 登录注册视频分屏与切换稳定性：认证页改为左侧全高品牌视频、右侧窄版高卡片，分栏交界使用向暗色工作区延伸的渐隐遮罩，并复用 Aceternity `CometCard` 与 `SpotlightSurface` 的指针倾斜、高光和回弹；登录和注册提升到同一共享路由布局，通过卡片内 Ant Design Tabs 切换真实 `/login`、`/register` 路由并保留查询参数，不再并行保留底部文字入口。注册内容完整撑开卡片，短视口由右侧区域滚动。认证卡片固定使用项目暗色 Ant Design 主题，并覆盖浏览器自动填充颜色，避免浅色输入内层和不可见链接。需验证视频自动静音循环、海报降级、渐隐边界、Comet 移入移出、Tab 点击及键盘切换、前进后退、自动填充、注册按钮及 Linux.do 动态入口、短视口滚动、375/768/1200/1440px、减少动态效果和网络视频加载失败。

# 功能介绍

本文档记录当前项目已经实现的主要功能。

## 画布项目

- 支持创建多个画布项目。
- 支持项目重命名、删除、批量选择和批量删除。
- 支持单个画布项目导出为 JSON，也支持从 JSON 导入画布。
- 登录账号后，画布项目会同步到后端 `canvas_projects`；浏览器本地仍保留 `localForage` 缓存和未登录/后端不可用时的降级存储。

## 无限画布

- 支持拖动画布、滚轮缩放、缩放滑杆和重置视图。
- 支持小地图定位，可开关小地图。
- 支持点阵、网格线、空白三种背景。
- 支持浅色和深色主题。
- 支持框选、多选、全选、取消选择、删除选中。
- 支持复制粘贴节点和节点之间的连线。
- 支持撤销和重做节点、连线、视口、背景和助手会话变化。
- 支持节点连线，并高亮当前节点相关的上下游节点和连线。
- 支持快捷键帮助，覆盖缩放、框选、全选、复制粘贴、撤销重做、删除、退出选择和拖入图片。

## 节点

目前画布中有六类基础创作节点：

- 图片节点：展示上传图片、生成图片或我的素材图片。
- 文本节点：保存提示词、说明文案、AI 文字回答等文本内容。
- 分镜脚本节点：以结构化镜头表保存时长、画面、台词、景别、情绪、光影、音效以及图片/视频提示词，支持 AI 生成、全屏编辑、行级连接和批量生成分镜图或视频。
- 生成配置节点：汇总上游文本和图片，统一配置模型、比例、数量后批量生成图片或文本。
- 视频节点：展示上传视频或生成视频。
- 音频节点：展示上传音频或生成音频。

节点支持：

- 拖拽移动。
- 四角缩放。
- 图片节点等比缩放或自由比例切换。
- 查看节点基础信息和 JSON。
- 删除、复制、粘贴。
- 通过左右连接点建立上下游关系。

## 图片工作流

- 支持上传图片到新节点。
- 支持拖拽图片文件到画布。
- 支持替换已有图片节点内容。
- 支持下载图片节点。
- 支持把图片节点保存到“我的素材”。
- 管理员启用 OSS 后，新上传或生成的图片会通过后端写入 OSS，并以 `resource:<id>` 作为长期资源引用。
- 支持图片裁剪，并把裁剪结果生成为新的图片节点。
- 支持本地多角度变换，并把结果生成为新的图片节点。
- 支持生成失败后重试。
- 批量生成多张图片时会先展示为图片组节点，支持叠卡预览、展开查看全部结果并设置主图。

## AI 生成

画布生成主链路已经收敛到后端异步任务中心：前端读取浏览器本地保存的 Base URL、API Key 和 Model 配置，创建 `canvas_text`、`canvas_image`、`canvas_video` 或 `canvas_audio` 任务；后端队列负责调用 OpenAI 兼容接口、轮询 provider 结果、持久化任务结果，再由前端把结果写回画布节点。

后端任务 worker 支持的 OpenAI 兼容图像和文本接口：

- `/v1/images/generations`：文生图。
- `/v1/images/edits`：图生图/参考图编辑。
- `/v1/responses`：文本问答、带图问答和在线 Agent 工具调用。
- `/v1/models`：读取模型列表；火山方舟 Agent Plan 专属 `/api/plan/v3` 当前未提供 OpenAI `/models` 模型列表接口，需要手动填写模型名。

后端视频任务支持两类接口：

- OpenAI 风格视频：`POST /v1/videos`、`GET /v1/videos/{id}`、`GET /v1/videos/{id}/content`。
- 火山方舟 Agent Plan / Seedance 2.0：Base URL 使用 `https://ark.cn-beijing.volces.com/api/plan/v3`，创建任务为 `POST /contents/generations/tasks`，查询任务为 `GET /contents/generations/tasks/{id}`，成功结果读取 `content.video_url`。

Base URL 如果已经以 `/v1`、`/api/v3` 或 `/api/plan/v3` 结尾，系统不会再追加 `/v1`。因此 cpa 反代或火山方舟 Agent Plan 可以继续通过现有 Base URL + API Key + Model 方式配置，不需要新增火山生图 Provider。

配置弹窗里的“拉取模型列表”会尝试真实请求 OpenAI `/models`，不会为 Agent Plan 伪造模型结果。如果火山方舟 Agent Plan 返回 404，请手动增加 `doubao-seedance-2.0` 或文档列出的其他模型名。

可配置项：

- Base URL。
- API Key。
- 默认模型。
- 图片质量。
- 图片比例。
- 生成数量。
- 视频模型。
- 视频比例、清晰度、时长、生成声音和水印。

普通图片/文本节点可以直接输入提示词生成结果。生成配置节点可以读取上游节点内容，并按节点自己的配置批量生成多个图片或文本结果。生成配置节点支持预览当前提示词和参考图输入，并调整输入顺序。生成执行期间，前端不再直接长时间占用 provider 请求，而是轮询后端任务状态。

视频生成可从文本节点读取 prompt，从图片节点读取参考图，从视频节点读取参考视频，从音频节点读取参考音频。Seedance 2.0 支持最多 9 张参考图、3 个参考视频、3 个参考音频；分辨率支持 `480p`、`720p`、`1080p`（fast 模型不支持 `1080p`），比例支持 `16:9`、`4:3`、`1:1`、`3:4`、`9:16`、`21:9`、`adaptive`，时长支持 4-15 秒或智能时长。生成成功后会把视频插入画布为视频节点并使用原生播放器预览。参考视频和参考音频优先使用公网可访问 URL；本地素材会随任务序列化给后端，再由后端提交给兼容接口，是否支持取决于具体上游。

## 异步任务中心

新增 `backend/` 后端服务，提供任务、会话、消息、任务日志、上传文件和结果表。任务中心支持：

- 创建异步生成任务。
- 查询任务列表和单个任务状态。
- 失败任务重试。
- 运行中任务取消。
- 查询任务日志。
- 持久化任务结果，并为后续写回画布保留结果数据。

本地开发时，前端 `/api` 会代理到 `http://127.0.0.1:8080`。生产环境可通过 `VITE_CANVAS_BACKEND_URL` 指向任务中心后端。

## Agent 会话层

后端提供类似 LibTV 的会话接口：

- `POST /api/sessions`：创建 Agent 会话，提交自然语言创作指令、项目 ID 和画布快照。
- `GET /api/sessions/:id`：查询会话、消息、任务和结果。
- `POST /api/files`：上传会话参考素材。
- `GET /api/sessions/:id/results`：下载会话结果。
- LibTV 风格别名：`POST /api/create_session`、`GET /api/query_session/:id`、`POST /api/upload_file`、`GET /api/download_results/:id`。

Agent 会话会把创作指令、特殊要求和当前画布图片节点的资产标签摘要提交给后端，生成剧本、角色、场景、风格板、参考素材组、分镜、镜头配置和成片节点。

管理员后台可维护影视分镜提示词模板版本，同一时间只启用一个版本。模板支持插入剧情、用户要求和画布资产变量；新部署会初始化默认影视分镜提示词。

## Agent 影视工作流

通过 Agent 会话输入剧情、短剧/MV/TVC 目标或品牌 brief 后，系统会调用文本模型，结合当前画布中图片节点的资产标签，生成：

- 剧本节点：保存项目标题、故事梗概和原始 brief。
- 角色节点：保存模型抽取的角色、一致性描述和表演方向。
- 场景节点：保存模型抽取的空间、时间、氛围和调度信息。
- 风格板节点：保存整体摄影、光线、色彩、质感和一致性规则。
- 参考素材组节点：汇总当前画布可复用资产及标签，例如角色、环境、武器、道具。
- 镜头配置节点：每个镜头生成可直接用于视频模型的提示词，包含主体动作、景别、构图、机位、运镜、光线、色彩、风格和一致性约束。
- 成片节点：作为多个镜头结果汇总和版本回填入口。

图片节点详情中可以维护资产标签，例如 `角色: 张三`、`环境: 大厅`、`武器: 镇邪剑`。分镜模型会把这些标签作为剧情拆解和镜头提示词的一部分；命中标签的图片节点会自动连接到对应镜头配置节点，并以 `@[node:id]` 形式进入生成上下文。

Agent 影视工作流仍可生成文本、图片、视频和生成配置节点；手工分镜流程还可使用独立脚本节点统一维护结构化镜头行。两条流程都会在 metadata 中记录 `workflowKind`、`workflowTitle`、`shotIndex`、`videoEditOperation` 等业务语义。

## 视频编辑闭环

任务中心支持以下视频编辑任务语义：

- 文生视频。
- 图生视频。
- 视频续写。
- 视频局部修改。
- 元素替换。
- 镜头/运镜调整。
- 风格迁移。
- 参考音频生成视频。
- 结果版本对比。

画布里的生成配置节点切到“视频”模式后，也可以选择上述视频操作；如果没有手动选择，系统会根据连接的参考图、参考视频或参考音频推断默认操作。

## 画布助手

画布右侧助手面板支持：

- 文本问答。
- 生图。
- 读取当前选中节点作为引用。
- 自动把选中节点的上游节点也纳入引用。
- 粘贴图片到助手输入框并插入画布。
- 历史会话。
- 删除单条或多条会话。
- 重试回答。
- 把助手生成的文本插入画布。
- 把助手生成的图片插入画布。
- 折叠和展开助手面板。

## 素材

“我的素材”是账号素材库，登录后会同步到后端 `assets`，本地 `localForage` 作为缓存和降级存储，支持：

- 新增文本素材和图片素材。
- 编辑素材标题、封面、标签、来源、备注和内容。
- 删除素材。
- 按关键词搜索。
- 按类型筛选。
- 分页浏览。
- 复制文本素材。
- 下载图片素材。
- 从画布节点加入素材。
- 在画布中插入素材。

## 配置和同步

- 支持账号登录和管理员后台，管理员可维护系统渠道、影视分镜提示词模板和阿里云 OSS 配置。
- 登录账号后，画布项目和“我的素材”会同步到后端；图片、视频、音频等二进制资源在 OSS 启用时通过后端上传到 OSS。
- 配置与用户偏好弹窗支持用户自定义渠道、系统渠道模型选择、默认模型和生成偏好。
- 画布生成、异步任务中心和 Agent 会话由 `backend/` 服务承载。

## 当前限制

- OSS 未启用或配置不可用时，图片、视频、音频会继续降级保存在当前浏览器本地；这种降级资源不能跨浏览器或跨设备可见。
- 删除画布节点或素材时，当前只清理本地降级 Blob 或移除 JSON 引用，不会自动删除 OSS 远端对象。
- 系统渠道的真实 API Key 只保存在后端。通用画布生成和部分手动任务仍可能使用浏览器本地自定义渠道并随异步任务提交给本机或自部署后端队列，只适合个人或可信环境使用。
- Seedance 本地参考视频/音频更建议使用公网可访问 URL；上游是否接受前端传入的本地数据取决于具体兼容接口。
- Seedance 返回远程视频 URL 时，前端会尽量下载为本地 Blob 持久化；如果因 CORS 或网络限制无法下载，会保留远程 URL，后续是否可播放取决于上游 URL 的有效期。
- 画布更适合桌面端使用，移动端触控体验还未系统完善。

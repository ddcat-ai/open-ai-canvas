# Render 部署

在 Render 中导入你自己的项目仓库即可部署。

## 部署步骤

1. 登录 Render。
2. 新建 Web Service，并选择你的项目仓库。
3. 确认部署配置。

部署完成后，打开 Render 分配的 `.onrender.com` 域名即可访问。

## 免费版说明

默认使用 Render 免费 Web Service：

- 空闲约 15 分钟后会休眠，下次访问会自动唤醒。
- 登录账号后的画布、素材、任务和 Agent 会话需要后端任务中心承载；媒体资源跨设备读取需要管理员配置 OSS。
- 适合体验和演示，不适合长期保存正式数据。

长期使用建议优先部署到 Vercel，并为后端配置持久化数据库和 OSS。
